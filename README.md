# 💱 Currency Converter

A responsive currency converter web application built using React. It allows users to convert between different currencies in real-time using exchange rate data from a public API.

---

## 🚀 Features

### 💱 Currency Conversion

* 🔄 Convert between multiple currencies
* 📊 Real-time exchange rates using public API
* ⚡ Instant conversion calculation

### 🔁 Smart Interactions

* 🔄 Swap currencies with a single click
* 📥 Dynamic currency options from API
* 🎯 Controlled input for accurate calculations

### ⚛️ React Features

* 🧠 Custom hook (`useCurrencyInfo`) for fetching API data
* 🔄 State management using React hooks
* 📦 Reusable components (InputBox)

### 🎨 UI/UX Features

* 🖼️ Background image with blur effect
* 📱 Responsive design
* ⚡ Clean and smooth interface

---

## 🛠️ Tech Stack

* React.js
* Vite
* Tailwind CSS / CSS
* Public Currency API

---

## 💻 Installation & Setup

### 1. Clone the repository

git clone https://github.com/rachitgupta05/currency-converter.git

cd currency-converter

### 2. Install dependencies

npm install

### 3. Run the project

npm run dev

---

## 🧠 How It Works

1. User enters an amount and selects currencies
2. App fetches exchange rates using a custom hook
3. Conversion is calculated instantly
4. User can swap currencies with one click

---

## 🔐 Data Handling

* Uses a public API (no API key required)
* No sensitive data is stored
* No backend required
